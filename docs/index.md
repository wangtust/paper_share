# Paper Share

这里是三人共同维护的论文阅读分享站点。

我们用这个仓库记录：

- 不同研究领域的重要论文
- 每个人的论文阅读笔记
- 领域阅读路线和关键问题
- 可复用的引用、图表和实验线索

## 推荐阅读方式

如果想快速了解最近读了什么，先看 [总论文清单](reading-list.md)。

如果想按方向系统阅读，先看领域页：

- [大语言模型](topics/llm.md)
- [RAG](topics/rag.md)
- [Agent](topics/agent.md)
- [多模态](topics/multimodal.md)
- [计算机视觉](topics/cv.md)
- [自然语言处理](topics/nlp.md)
- [系统与工程](topics/systems.md)

如果要新增论文笔记，复制 [论文笔记模板](note-templates/paper-note.md)，再放到对应领域目录下。
